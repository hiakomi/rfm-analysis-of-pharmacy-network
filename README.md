# RFM-анализ клиентской базы
Комплексный анализ клиентского поведения на основе транзакционных данных с применением RFM-сегментации.

## О проекте
В данном проекте проводится исследование клиентской базы аптечной сети с целью выявления различий в покупательском поведении.

Идея анализа основана на том, что клиенты существенно отличаются по активности:
часть из них совершает регулярные покупки и формирует основную выручку,
в то время как значительная доля пользователей постепенно снижает активность и уходит.

Цель проекта — сегментировать клиентов на основе их поведения, определить ключевые группы с точки зрения ценности и риска оттока, а также сформировать рекомендации по работе с ними.

## Основные вопросы
В рамках анализа исследуются следующие вопросы:
1. Как распределяются клиенты по давности, частоте и сумме покупок?
2. Какая доля клиентов генерирует основную выручку?
3. Какие сегменты являются наиболее ценными для бизнеса?
4. Какие группы клиентов находятся в зоне риска оттока?
5. Какие сегменты имеют потенциал роста?

##  Используемые данные
В анализе используются транзакционные данные аптечной сети, включающие:
- идентификаторы клиентов
- информацию о покупках
- суммы транзакций
- временные метки операций

Данные предварительно были очищены и агрегированы до уровня «один клиент — одна запись».
Подробная реализация каждого шага представлена в файле `rfm_analysis.sql`.

## Ключевые инсайты

1. Клиентская база неоднородна: небольшая группа клиентов демонстрирует значительно более высокую активность
2. Распределение покупок имеет выраженный «длинный хвост» — большинство клиентов покупает редко
3. Значительная часть выручки формируется за счет наиболее активных и ценных клиентов
4. Новые клиенты быстро переходят в менее активные сегменты без дополнительного удержания
5. Частота покупок сильнее влияет на ценность клиента, чем разовые крупные транзакции
6. Существует выраженный потенциал для реактивации части клиентской базы

## Выводы

Поведение клиентов существенно различается и требует сегментированного подхода.

Ценность клиента определяется сочетанием:
1. давности последней покупки
2. частоты взаимодействия
3. суммарного объема покупок

В рамках анализа была проведена RFM-сегментация клиентской базы, выделены ключевые группы, а также разработаны рекомендации по работе с каждым сегментом — от удержания наиболее ценных клиентов до реактивации пользователей с риском оттока.

Полученные результаты могут быть использованы для построения маркетинговых стратегий, направленных на повышение LTV, снижение оттока и оптимизацию коммуникаций с клиентской базой.
