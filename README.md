# RFM-анализ клиентской базы

## О проекте

В данном проекте выполнен RFM-анализ клиентской базы на основе транзакционных данных.

Цель — сегментировать клиентов по их покупательскому поведению и выявить группы с различной ценностью для бизнеса.

## Задачи

- Рассчитать метрики:
  - Recency (давность последней покупки)
  - Frequency (частота покупок)
  - Monetary (сумма покупок)

- Сегментировать клиентов на группы
- Проанализировать распределение клиентов по сегментам
- Сформулировать бизнес-выводы

## Используемые инструменты

- SQL (PostgreSQL)
- Metabase (визуализация)

## Логика расчёта

Анализ выполнялся в несколько этапов:

1. Очистка и подготовка данных  
2. Расчет R, F, M метрик  
3. Разбиение на квантили  
4. Присвоение RFM-сегментов  
5. Агрегация и анализ сегментов  

Подробная реализация представлена в файле `rfm_analysis.sql`.

## Ключевые инсайты

- Небольшая доля клиентов генерирует основную выручку
- Значительная часть клиентов относится к «спящим»
- Сегмент лояльных клиентов стабильно активен
- Есть потенциал для реактивации клиентов

## Бизнес-выводы

- Фокус на удержании лояльных клиентов  
- Разработка кампаний для реактивации  
- Персонализация предложений
